# 🛒 Amazon Website Clone

A responsive Amazon Website Clone built using HTML and CSS. This project recreates the basic layout and design of Amazon's homepage, providing hands-on experience with modern web development concepts and responsive user interface design.

## 📌 Project Overview

The Amazon Website Clone is a front-end web project designed to mimic the appearance and structure of Amazon's homepage. It includes a navigation bar, search section, hero banner, product categories, and footer sections, offering a realistic e-commerce website experience.
